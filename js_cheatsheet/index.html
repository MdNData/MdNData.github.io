In the script.js file

console.log()

Show what's within the brackets in the terminal.

console.log('Hi!'); // display -> Hi!

// and /* */

They are used to describe what a piece of code is for, useful both for us and for other programmers, a single comment line is made with //, instead more lines of comment are made with /* comments */, you can also put a comment in the middle of an instruction like this :

console.log (/* comment */ 18); // display -> 18

JavaScript + HTML

< script >

Script tags allow you to insert JS code into our html.file exactly like <style> does for css.


Obviously it is better to write the JS code in an external file, thas is then integrated in html with :

<script src="path/script.js" type="text/javascript"> </script>

When we insert a script in the html file, based on where we insert it the page will stop loading to load the script, once the script is loaded, the page will resume loading, if we put the script at the bottom of the page, it will load first the page and then the scripts, if we put the script in the middle of page, the first half of the page will be loaded up to the script, then the script will be loaded and once loaded, the page will resume its loading.

╚> defer

Specifies that the script is loaded only after the entire page has loaded.

<script src="path" type="text/javascript" defer > </script>

╚> async

Attribute that allows you to load the scripts in the meantime that the html page is also loading (load the script in the background) without stopping anything, the script will be executed once downloaded no matter if the page has already loaded or not.

<script src="path" type="text/javascript" async > </script>

The DOM

document

The door to access the DOM, before changing any html element you have to select it via document.

document.

document.tag.innerHTML = " - "

It allows you to access the contents of a tag and set it like this:

document.body.innerHTML = "<h1> Hello World! </h1>"
//In this way we can access the body tag, delete everything inside it and replace it with the header which contains "Hello World!"

document.querySelector(' - ')

Allows you to select more specific elements through CSS selectors and returns the first matching element found in the DOM, you can select tags, .classes and #id.

document.querySelector('p');
//Select the first <p> it finds

document.getElementById(' - ')

It is used to select a certain id.

document.getElementById('someId');
//Find the element with this id and select it

.style.property = 'attribute'

It is used to change the style of an element, first you select the element, then you add .style, then you decide which style to change, for example .fontSize and finally you assign the value = "12px".

document.querySelector('p').style.color = "#fff";
//In this case we select the first <p> we find on the page and change its color.

//Or

let firstP = document.querySelector('p');
firstP.style.color = red;
//In this case we assign to a variable the selection of <p>, and then through the variable we change its style.

.createElement = 'tag'

It creates an element based on the specified tag, but this does not appear in the document because it is empty and with no innerHTML, furthermore to show it on the page it is necessary to append it to another element.

let newP = document.createElement = 'p';
//A new element has just been created!

.appendChild()

It is used to add the element created to the page, assigning it as a child to an html element that already exists (it will be assigned as the last child of that element).

let newP = document.createElement = 'p';
//Create the element
newP.id="info";
//Assign it an id
newP.innerHTML = 'The text inside the paragraph';
//Add some text to it
document.body.appendChild(newP);
//And finally add it to the body as its last element

.removeChild()

Similar to .appendChild() only that instead of adding, it removes.

.hidden = true/false

Hide the selected HTML element.

newP.hidden = true;
//with true the element be hidden

.onclick = function;

We can add interactions with the DOM via events.


.onclick allows you to add a function to be started when interacting with an element via a click.


Try clicking on the word JavaScript in the top navigation bar! This happens thanks to onclick!

let element = document.querySelector("button");
// select the element

function turnButtonRed (){ ... }
// make a function to change the style of selected element

element.onclick = turnButtonRed;
// create the event and wait for it to happen

.addEventListener( 'event', function );

This function waits for a specific event to happen and starts the assigned function, takes 2 arguments, the event type as a string and the event hadler function.

let element = document.querySelector("button");
// select the element

function turnButtonRed (){ ... }
// make a function to change the style of selected element

element.addEventListener('click', turnButtonRed);
// create the event and wait for it to happen
