Manichiura

Ingrijirea si infrumusetarea unghiilor este un ritual la care nicio femeie ar trebui sa renunte.

Vezi lista preturi
 

Lista Preturi

Constructie unghii gel : 80 ron

- pretul va creste in functie de lungimea unghiilor -

Intretinere unghii gel :

- unghii scurte / medii cu model   inclus pe 2 unghii 70 ron

- unghii lungi cu model inclus        pe 2 unghii 90 ron

- intretinere unghii aplicate in    alt salon 90 ron

Aplicare gel pe unghii naturale :

- unghii scurte / medii cu model   inclus pe 2 unghii 75 ron

- unghii lungi cu model inclus        pe 2 unghii 90 ron

Aplicare oja semipermanenta :

- Mani - model inclus pe 2 unghii 65 ron
