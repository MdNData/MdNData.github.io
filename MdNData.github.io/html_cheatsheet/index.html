Setting HTML.html file

< !DOCTYPE html >

The first line of the file.html must contain this line of code, it is useful to make the browser understand that the HTML language is being used and in particular the latest standard released.

< !DOCTYPE html >
.
..
...

< html >

It is used to create the structure of the html page, everything inside these tags will be interpreted as html text, also known as the root element.

< !DOCTYPE html >
 <html>

</html>

╚> lang="en"

To tell the browser what language our content is displayed in, we need to add lang="" in the html tag : < html lang="en" > (if the content is in English).

< head >

It must be added just below the html tag, it contains the metadata of the page: technical declarations, not always actually visible on the page.

< !DOCTYPE html >
 
 <html lang="en" >  
    <head>

    </head>
</html>

╚> < meta />

They are used to add information about the page.

╚> link to all meta

Here.
A list of all the meta (or almost) that can be used.

╚> < meta charset="UTF-8" >

It is inserted in all the sites, it is used to tell the browser that we will use all the existing fonts, from English to Chinese etc.

╚> < meta name='keywords' content="" />

It contains the words that mainly characterize the page, about 13 are recommended and all divided by commas.

╚> < meta name='viewport' content='width=device-width, initial-scale=1.0' />

It is used to specify to the browser that our site will be responsive, and renders it consequently.

╚> < meta http-equiv='X-UA-Compatible' content='ie=edge' />

It tells the browser that our site is compatible with edge.

╚> < link rel='icon' type='image/png' href=" " />

It is used to insert a small image at the top of the page in the browser, a size of 64x64px is recommended.

╚> < title >

It is used to give a name to our page, this name will be displayed at the top of the browser tab, and is inserted between the head tag.

╚> < style >

Allows the addition of CSS code inside this tag.
CSS is used to change the style of HTML elements.

╚> < link href="path" rel="stylesheet" type="text/css" >

Import an external CSS file, in order to have the page styles organized in external files, we can import as many as we want.

╚> < script >

Allows the addition of JavaScript code inside this tag.
JS is used to make HTML elements interactive.

< body >

The container of everything that appears on your page. It is always placed under the head tag. All the HTML tags listed below will be contained within this tag.

╚> < nav >

The nav is used for link blocks such as menus and content tables, it can also be placed inside other tags, or it can be used simply by yourself.

╚> < header >

Specially created to contain the main title tags.

╚> < main >

Helps SEO and programmers to find the main part of your page, neither the footer nor the nav should be placed inside the main but the headers can.

╚> < section >

One of the most used tag, very usefull to sectionide the page in smallest sections with elements that have the same use or purpose.

╚> < article >

Article contains parts that make sense for themselves, detached from the main content, can be inserted inside section or vice versa.

╚> < aside >

They are used to group additional information such as: bibliographies, Endnotes, Comments, Pull quotes, Editorial sidebars.

╚> < figure > and < figcaption >

Figure is an element used to enclose media such as images, illustrations, diagrams and code that refers to the main content of the page.

Figcaption fits into figure, is used as a visible description of the media in it.

╚> < footer >

It is used to enter final information such as email, site map, phone number etc.

< h1 - h2 - h3 - h4 - h5 - h6 >

They are the titles of the page that indicate the start of a new content, starting from h1 the largest title (there should be only / at least one) to h6 the smallest one. They are very important elements both for the reader and for the browser that uses them for the search, if used well they can make your web-site appear first in the search engines.

< p >

They are simply pieces of text. Very useful, it is recommended to use multiple paragraphs instead of using just one creating too complex sentences, it also helps search engines.

< span >

They contain short pieces of text or other html, used to separate small pieces of content that are on the same line as other text without affecting the display of the tag that contains it.

< br />

Stop the current line of text and start a new
one in the next line below.

< hr />

Creates a separator line as long as the full size of the box that contains it, often used to separate content.

< strong > - < b >

strong is used to enlarge the text but also to emphasize it not only for the reader but also for the browsers, b instead only enlarges it.

< em > - < i > (Italic)

em is used to make italic text but also to emphasize it not only for the reader but also for the browsers, i instead only make it italic.

< u >

Pull a line below the text, usually specifying errors, or changes to a text also emphasizing it for the browser.

< s >

Specifies that what is inside this tag is a piece of text that is no longer valid.

< code >

Specifies that what is inside this tag is code.

< samp >

Specifies that what is inside this tag is code, but coming from another computer or source of code.

< kbd >

Specifies that what is inside this tag is code that need to be entered by the user such as "ctrl + c"

< var >

Specifies that what's inside this tag is a variable.

< pre >

All the text in this tag will be reported with the same spaces, without removing any space as happens with the other text-tags.

title = ' - '

Allows you to view a text by holding the mouse over an element on the web page for a few seconds

id = ' - '

They are useful for better specifying what an element consists of, giving it a unique identity, it is also used to embellish and select it with css and js.

< a href="path" target="_blank" />

It is used to create a web link between your page and an external page, or to internal sections of your page.
It has 2 peculiar attributes :

- href attribute is used to specify which page or which part of it to go to, the link of the external page or the id of the element to connect to must be placed inside href.

- target attribute, it has only one value "_target" and is only used if we want to open the link in a new web page.

- href = " # " it is used to create a dead link, useful if we do not yet know where the link should connect.

- Each html element can be transformed into a link just insert it between the closing and opening tags of <a> element </a>.

< img src="path" alt="desription" />

This element is used to add images to the page.
It has 2 required attributes :

- src attribute is used to specify where the image we want to view is located, whether it is on the PC or on an online server.

- alt attribute, it is used to describe an image, and if the image is not loaded make this text appear in its place, and above all it improves the site making it more visible among the search engine results.

< progress min="" max="" value="" >

This tag provides the ability to create progress bars.
min and max are used to set a starting point and an ending point that marks the end of the task.
value indicates how far the upload has come.

< ul >

It creates an unordered list with circles.

╚> < li >

The li items are the data of the list, without them the list would not make sense to exist.

< ol >

It creates an ordered list with numbers.

< ul >

It creates an unordered list with circles.

╚> < li >

The li items are the data of the list, without them the list would not make sense to exist.
