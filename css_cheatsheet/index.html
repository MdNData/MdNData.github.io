In the style.css file

style = " - "

You can change the appearance of an HTML tag with style = " - " applied inside the opening tag.

<p style="color:red"> text <p>

< style >

Another way to insert styles inside the html file is to create a <style> tag inside the <head> tag of the body.

<style>
    p {
        color: blue;
    }
</style>

style.css

The last and also the best way to insert style is to write it in an external style.css file and insert it in the head of the html file.

<link href="path/style.css" rel="stylesheet" type="text/css">

tag selector

To edit HTML elements via CSS, we must first select them.
The least specific way to select elements is selecting them by tag, just rewrite them in the css file as follows:

p {
    //In this way we select all the p tags
}

p, div {
    //In this way we select all the p and div tags
}

class selector

To select a class you need to add a period in front of its name.
Selecting the class will also override the common CSS rules created by the tag selector.

.name {
    //In this way we select all the elements with name as class
}

tag.class

We can also select only certain elements that have a certain class.

div.className {
    //In this way we select only divs with className
}

accurate selection

We can make the selection more accurate by simply adding more selectors one after the other, if we want to select only the p's that are inside the divs we can do:

div p {
    //In this way we select all the p's in divs, works with tags, classes and ids
}

It also works with the > symbol

div > p {
    //In this way we select all the p's in divs
}

adjacent selection

To select only the element after a certain element:

form label + input {
    //it will select only the first input that it finds after label inside the form, if the element after label is not an input it will not select anything.
}

[attribute=value]

Select only the elements that have that attribute with that value.

[type='radio'] {
    //it will select only elements with type attribute that have a 'radio' value.
}

ID selector

To select a ID you need to add a # in front of its name.
Selecting the ID will also override the common CSS rules created by tag and class selectors.

#name {
    //In this way we select the element with name as id
}

!important

If we want at all costs that our element is not modified by other selectors, we can add !important at the end of the CSS style we want to apply.

div {
color: red !important;
    //The divs will always have the text color red!
}

:root selector

Select the container of the entire html file, usually css variables are assigned to it to make them globally accessible.

:root {

}

--name : value;

By creating a CSS variable, we can reuse it anywhere in the code, so if we need to change its value just change it in the variable.
Without variables we would have to look in the code for all the places where we have to change the same value over and over again, wasting a lot of time.
Variables are usually declared inside the selector :root {} in order to have them globally accessible.

//declare the variable
:root {
     --red-color : red;
}

//use it and also add an emergency value if the variable is not available
p {
     color: var(--red-color, red);
}

background: linear-gradient(to top, color % %)

It is used to create a colored background that changes color according to the position we define.


It should be assigned to the background or background-image property.


We can assign more than one to the same background.


Inside linear-gradient () we first define where the color should start from, we can use deg (0-360deg) or fixed positions like "to top", "to bottom", "to right" or "to left".


After that we insert the colors we want to give to the background, we can say exactly at which % to start or to finish, the first % is the starting one,the second % is the end one.

background: linear-gradient(to top, red 0% 25%, blue 25% 50%, green 50% 100%);
